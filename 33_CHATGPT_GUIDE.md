# Purpose

This guide helps UK charities use ChatGPT safely, confidently and proportionately.

It focuses on practical day-to-day use rather than technical detail.

This guidance should always be read alongside:

- Before You Paste Framework
- GDPR Decision Rules
- Safeguarding Guidance
- AI Tool Evaluation Framework

# What is ChatGPT?

ChatGPT is a conversational AI assistant that can help with:

- writing
- editing
- brainstorming
- summarising
- planning
- coding
- research support
- translation
- accessibility improvements

It should support people, not replace professional judgement.

# Best Uses for Charities

Suitable examples include:

- website copy
- newsletters
- social media
- funding application first drafts
- meeting agendas
- board paper structure
- policy first drafts
- volunteer resources
- training material
- FAQs
- plain English rewrites

# Green Uses

Use ChatGPT with:

- public information
- fictional examples
- generic templates
- published reports
- brainstorming
- grammar and editing

# Amber Uses

Only with organisational approval where appropriate:

- anonymised internal reports
- internal planning
- draft strategies
- aggregated survey findings
- unpublished content

Always apply meaningful human review.

# Red Uses

Do not paste into public AI tools:

- safeguarding records
- identifiable beneficiary information
- HR case files
- payroll
- donor databases
- passwords
- API keys
- legal advice
- confidential contracts
- medical information

# Human Review

Always check:

- factual accuracy
- bias
- tone
- accessibility
- copyright
- references
- organisation-specific details

AI output is a draft, not a final answer.

# Prompting Tips

Good prompts:

- explain the audience
- explain the goal
- provide context
- request plain English
- ask for structured output
- request limitations

Avoid vague prompts.

# Common Mistakes

- trusting answers without checking
- uploading sensitive data
- assuming AI knows charity policy
- copying text without review
- using outdated information

# Governance

Organisations should define:

- approved accounts
- approved use cases
- review requirements
- escalation routes
- training expectations

# Monthly Review

Review when:

- OpenAI changes major features
- privacy terms change
- pricing changes
- regulator guidance changes

# Success

This guide succeeds when charity staff understand how to use ChatGPT safely, know its limitations and recognise when human judgement is essential.
