# Purpose

This guide explains how UK charities can use Notion AI safely, responsibly and proportionately.

It focuses on using Notion AI to improve organisational knowledge, documentation and collaboration rather than replacing professional judgement.

# What is Notion AI?

Notion AI is built into the Notion workspace and can assist with:

- writing
- summarising
- brainstorming
- meeting notes
- project documentation
- knowledge management
- task planning
- document editing
- translation

# Best Uses for Charities

Suitable uses include:

- internal documentation
- project plans
- meeting agendas
- action summaries
- knowledge bases
- onboarding guides
- policy first drafts
- training notes
- process documentation
- wiki content

# Green Uses

Generally suitable:

- public information
- fictional examples
- generic templates
- public policies
- brainstorming
- formatting
- plain-English editing

# Amber Uses

Use additional care with:

- internal project documentation
- unpublished strategies
- anonymised reports
- internal meeting notes
- board papers
- operational documentation

Require:

- approved workspace
- appropriate permissions
- human review
- manager approval where appropriate

# Red Uses

Do not upload:

- safeguarding case records
- identifiable beneficiary information
- payroll
- HR investigations
- passwords
- API keys
- privileged legal advice
- confidential contracts
- medical information

# Knowledge Management

Notion AI works best when:

- documents are well organised
- pages are versioned
- outdated guidance is archived
- ownership is clear
- review dates are maintained
- evidence is linked

AI cannot compensate for poor knowledge management.

# Workspace Governance

Define:

- workspace owners
- editing permissions
- AI usage expectations
- document review process
- archive process
- approval workflow

# Human Review

Always verify:

- accuracy
- context
- tone
- legal wording
- accessibility
- links
- version numbers
- references

# Common Mistakes

Avoid:

- treating drafts as final
- storing sensitive information unnecessarily
- allowing outdated pages to remain live
- giving excessive edit permissions
- assuming AI understands organisational policy
