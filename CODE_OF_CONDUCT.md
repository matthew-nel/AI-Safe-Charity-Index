# Code of Conduct

AI Safe Charity Index is built around trust, practical learning and respectful collaboration.

Everyone who participates should be able to contribute without being dismissed, intimidated or exposed to unnecessary harm.

## Expected Behaviour

Participants should:

- communicate respectfully
- assume good faith where reasonable
- give constructive feedback
- explain technical terms
- protect privacy
- respect lived experience
- cite evidence where possible
- correct mistakes openly
- avoid personal attacks
- focus criticism on ideas and work
- follow safeguarding and data-protection rules

## Unacceptable Behaviour

Unacceptable behaviour includes harassment, discrimination, threats, intimidation, doxxing, publishing private information, sharing safeguarding information, sharing identifiable beneficiary information, deliberate misinformation, unsolicited sales, spam and personal attacks.

## Privacy and Safeguarding

Do not share personal data, safeguarding case information, identifiable beneficiary stories, employee records, donor records, passwords, API keys or confidential documents in project spaces.

If a contribution includes safeguarding information, stop before publishing it. Use the appropriate safeguarding route within the relevant organisation or statutory service.

## Reporting

Open a GitHub issue for ordinary project concerns. Do not use GitHub for urgent safeguarding reports or private personal-data incidents.
