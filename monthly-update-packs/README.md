# Monthly Update Packs

Store large Markdown research outputs here before Codex converts them into public documentation updates.

Create one folder per month:

```text
monthly-update-packs/
└── 2026-08/
    ├── 01_perplexity_before-you-paste.md
    ├── 02_chatgpt_before-you-paste_synthesis.md
    ├── 03_perplexity_gdpr-ai-charities.md
    ├── 04_chatgpt_gdpr-ai-charities_synthesis.md
    └── 99_codex_handoff.md
```

Raw research packs are working material. They should not be treated as public guidance until reviewed and converted into `docs/` pages.
