# Tools

This section covers tool evaluation and guidance for common AI products.

Tool guidance must be checked against current provider documentation before procurement or rollout. Product terms, retention settings, training settings and admin controls change frequently.

## Core Rule

Do not approve a brand name alone. Approve the **tool, account plan, settings and use case**.

For example, a personal paid account may not provide the same protections as a business or enterprise workspace.

## Every Tool Review Checks

- what the tool does
- plan and account differences
- whether inputs may be used for model training
- retention and deletion controls
- admin controls
- connectors and actions
- suitable charity uses
- Green, Amber and Red use cases
- human review requirements
- known limitations
