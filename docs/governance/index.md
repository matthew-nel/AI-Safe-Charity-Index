# Governance

Good AI governance should be achievable, proportionate and useful.

Trustees remain responsible for decisions made by their charity. AI can support drafting, research and analysis, but it should not be the sole basis for a material decision.

## Start With Minimum Controls

Small and medium charities should start with:

- a named owner for AI use
- simple staff rules
- an approved-tools list
- a basic AI-use register
- a tool approval process
- human review expectations
- an incident route
- trustee visibility over high-risk use

## Trustee Questions

Trustees can ask:

- What AI tools are staff already using?
- Are personal accounts being used for charity work?
- Which uses involve personal, confidential or safeguarding information?
- Who approves tools and use cases?
- What happens if AI output causes harm?
- How are important decisions checked by humans?
