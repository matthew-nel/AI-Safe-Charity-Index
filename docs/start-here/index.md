# Start Here

AI can help charities draft, summarise, plan and learn. It can also create privacy, safeguarding, accuracy and accountability risks when used without clear checks.

Start with three rules:

1. Do not paste personal, confidential or safeguarding information into an unapproved AI tool.
2. Use the least information needed for the task.
3. Keep a human responsible for the output and the final decision.
