# Roadmap

## Phase 1: Foundation

- Create the MkDocs site foundation.
- Publish the task-based navigation skeleton.
- Add repository governance files.
- Publish the first Before You Paste guidance.

## Phase 2: Core Guidance

- Expand Before You Paste.
- Add AI and data-protection guidance.
- Add safeguarding and trustee guidance.
- Add charity workflow pages.

## Phase 3: Tools and Templates

- Publish the tool evaluation framework.
- Add current tool guides.
- Add policy, register and review templates.

## Phase 4: Quality Gates

- Add frontmatter validation.
- Add link checking.
- Add review-date reporting.
- Publish evidence and claim registers.
