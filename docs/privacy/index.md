# Privacy

This page describes the current public website at a high level. It should be reviewed whenever the site adds analytics, forms, newsletter signup, embedded media, comments, search telemetry, a chatbot or any other feature that collects personal data.

## Current Site Position

As currently configured in this repository, the public site:

- is a static MkDocs site hosted on GitHub Pages
- does not include a contact form
- does not include analytics configured by this project
- does not include a newsletter signup
- does not include a chatbot
- does not intentionally collect personal data from visitors

GitHub and browsers may process technical information needed to serve the site. Review GitHub's own privacy information for platform-level processing.

## What Must Not Be Submitted Publicly

Do not submit personal data, safeguarding information, identifiable beneficiary stories, employee records, donor records, passwords, API keys, confidential charity information or raw research responses through public GitHub issues or public project spaces.

## Future Data Inventory

Before adding any interactive feature, complete a data inventory covering:

- controller identity and contact route
- hosting and security logs
- analytics and cookies
- contact forms or email
- mailing lists
- embedded media
- GitHub contribution workflow
- processors and international transfers
- retention
- individual rights and complaint route
- whether children may use the service
- whether any automated decision-making is involved

## Principle

The project should minimise personal data collection and keep research inputs separate from public guidance.
