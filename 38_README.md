# AI Safe Use Knowledge Base for UK Charities

Practical, evidence-led guidance to help charities use AI safely, confidently and responsibly.

> **Help charities save time with AI without risking people's data, trust or safety.**

## About the Project

The AI Safe Use Knowledge Base for UK Charities is an open, living resource for charities that want to use artificial intelligence responsibly.

It is designed primarily for small and medium-sized UK charities, while also providing deeper guidance for larger and more complex organisations.

The knowledge base translates complex guidance into practical steps that charity staff, volunteers, managers and trustees can understand and use.

2. **AI and Data Protection**  
   Understand personal data, special category data, anonymisation and DPIAs.

3. **Charity Workflows**  
   Find guidance based on the task you are trying to complete.

4. **AI Tool Guides**  
   Review current guidance for ChatGPT, Claude, Gemini, Copilot and Notion AI.

5. **Trustee Guidance**  
   Understand proportionate AI governance and oversight.

## Signature Framework

The defining feature of the project is the **Before You Paste** framework.

Before entering information into an AI tool, ask:

- Is personal data involved?
- Is the information confidential?
- Is safeguarding involved?
- Is the tool approved?
- Can the task be completed with less data?
- Could the output affect a person?
- Is meaningful human review in place?

The framework uses:

- **Green — Go ahead with normal checks**
- **Amber — Pause and check**
- **Red — Stop**

## How the Knowledge Base Is Maintained

The project uses a monthly research and release process.

New material may include:

- regulator guidance
- charity-sector reports
- AI provider documentation
- books and longform sources
- survey findings
- interview findings
- GitHub issues
- public feedback
- analytics and search behaviour

The process is:

```text
New evidence and feedback
        ↓
Claude Project research and analysis
        ↓
Updated Markdown and change reports
        ↓
Human review
        ↓
GitHub
        ↓
Automated site deployment
```

Claude supports research and drafting.

GitHub stores and publishes the approved public version.

Human review remains mandatory for material changes, especially those involving GDPR, safeguarding, employment, significant decisions or vulnerable people.

## Legal and Safety Notice

This resource provides practical guidance.

It is not:

- legal advice
- safeguarding advice for a specific case
- professional HR advice
- security assurance
- a substitute for a Data Protection Impact Assessment
- a replacement for organisational policy
- a substitute for specialist review

Seek appropriate professional support for complex or high-risk uses.
