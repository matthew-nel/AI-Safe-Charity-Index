# AI Safe Charity Index

An open-source, research-led knowledge base for safe, responsible and practical AI use in charities.

> Help charities save time with AI without risking people's data, trust or safety.

## About

AI Safe Charity Index is an open, living knowledge base for charities that want to use artificial intelligence responsibly.

It is designed primarily for small and medium-sized UK charities, while also giving larger organisations a clear starting point for governance, review and assurance.

## Start Here

The first public build focuses on:

- the Before You Paste framework
- plain-English AI safety guidance
- data protection and safeguarding routes
- trustee and governance guidance
- tool guidance for common AI products
- transparent release notes and review dates

## Technology

This repository uses:

## Project Specification Corpus

The numbered Markdown files in the repository root are the project blueprint. They contain the product vision, information architecture, publishing standards, evidence model, content specifications, governance model and draft page content.

Use those files to drive implementation decisions before adding or restructuring public pages in `docs/`.

## Local Development

```bash
pip install -r requirements.txt
mkdocs serve
```

Then open `http://127.0.0.1:8000`.
