# Purpose

Provide one consistent framework for evaluating every AI tool referenced by the knowledge base.

The framework compares tools against charity needs rather than marketing claims.

# Evaluation Principles

- Evidence-led
- Vendor neutral
- Plain English
- Updated regularly
- Charity-first
- GDPR-aware
- Human-centred

- Security notes
- Human review required
- Sources
- Last reviewed
- Next review

# Review Rules

Do not:

- recommend based on hype
- copy vendor marketing
- assume enterprise features exist on free plans

Always:

- cite official documentation
- identify uncertainty
- explain trade-offs
- review regularly
