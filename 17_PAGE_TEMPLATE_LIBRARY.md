# Purpose

This document defines the standard page templates used throughout the knowledge base.

## Standard Guidance Page

1. Page title
2. One-sentence summary
3. Quick answer
4. Green / Amber / Red guidance (where relevant)
5. What to do
6. Example
7. Why it matters
8. Advanced guidance (optional)
9. Related pages
10. Sources
11. Page metadata

## Tool Review

- Overview
- Charity use cases
- Benefits
- Risks
- GDPR considerations
- Recommendation
- Review dates

## Decision Guide

- Question
- Quick answer
- Decision tree
- Examples
- Next step

## Policy Template

- Purpose
- Scope
- Responsibilities
- Policy
- Review cycle

## Workflow Guide

- Goal
- Steps
- Checklist
- Common mistakes

## FAQ

- Question
- Short answer
- Link to detailed guidance

## Page Metadata

Every page should include:

- Version
- Last reviewed
- Next review
- Evidence level
- Audience
- Status

# Definition of Success

Readers should recognise the structure of every page and quickly find the answer they need.
